# vlr-multicharacter — Multicharacter

A multicharacter selection and creation screen in the Valora industrial-luxe style,
framework-aware and database-backed.

| | |
|---|---|
| **Framework** | Qbox / QBCore / ESX (auto-detected) |
| **Storage** | oxmysql |

## Features

* Character select / create / delete screen.
* Industrial-luxe NUI.
* Framework-aware character handling.
