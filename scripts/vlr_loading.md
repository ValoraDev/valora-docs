# vlr_loading — Loading Screen

A standalone, industrial-luxe loading screen for your server — themeable, lightweight and
with no dependencies.

| | |
|---|---|
| **Type** | Loading screen (standalone) |
| **Storage** | none |

## Features

* Clean Valora-style loading screen.
* Fully open `config.lua` and `html/` for branding, music and background media.
* No framework or database required.

## Dependencies

None. `vlr_loading` is standalone.
